# Jarvis AI Assistant 🤖

## Description
Jarvis AI Assistant is a Python-based voice assistant that can listen to user commands and perform automated tasks. The assistant activates when it hears the wake word **"Jarvis"** and can open websites, play music, and respond using text-to-speech.

## Features
- 🎤 Voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Opens websites using voice commands
- 🎵 Plays songs using voice commands
- ⚡ Wake word activation ("Jarvis")

## Technologies Used
- Python
- SpeechRecognition
- Pyttsx3
- Webbrowser
- Microphone input
